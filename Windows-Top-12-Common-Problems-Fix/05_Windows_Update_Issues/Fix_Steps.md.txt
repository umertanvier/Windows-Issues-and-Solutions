# Windows Update Issues

## Symptoms:
- Windows updates fail to download or install.
- Updates get stuck at a certain percentage.
- "We couldn't complete the updates" error message appears.
- The system is stuck on "Getting Windows Ready" or "Reverting Changes" screen.

## Most Important Fixes:

1. **Restart the Computer**:
   - Start with a simple **restart**. This can often resolve update issues, especially when updates get stuck. Restarting the system refreshes the network connection and resets any pending update processes.
   - After restarting, go to **Settings > Update & Security > Windows Update** and check for updates again.

2. **Check for Disk Space**:
   - Windows updates require sufficient free space on your **C:** drive (at least 5-10 GB free). If there’s not enough space, updates may fail.
   - Open **File Explorer**, right-click on the **C: drive**, and select **Properties** to check the available space.
   - If needed, run **Disk Cleanup** to delete temporary files and free up space:
     - Type **Disk Cleanup** in the Start Menu, select the C: drive, and click **OK**.
     - Select **Clean up system files**, check **Windows Update Cleanup**, and click **OK** to free up space.

3. **Manually Install Updates**:
   - If a particular update is failing, you can manually download it from the **Microsoft Update Catalog**.
   - Visit the [Microsoft Update Catalog](https://www.catalog.update.microsoft.com/Home.aspx), search for the update (e.g., **KB4537572**), download the installer, and run it manually.
   - This bypasses the regular update process and directly installs the update.

## Fix:
After restarting the system and freeing up space on the C: drive, updates were successfully installed. Manually downloading and installing the update from the **Microsoft Update Catalog** also resolved the issue when the automatic update failed.
