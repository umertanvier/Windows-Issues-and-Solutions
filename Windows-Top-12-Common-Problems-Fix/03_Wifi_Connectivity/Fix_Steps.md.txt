# Wi-Fi Connectivity Issue

## Symptoms:
- **Wi-Fi icon** is grayed out, showing no available networks.
- "No Internet" or "No Connectivity" message is displayed, even though the Wi-Fi is connected.
- The Wi-Fi keeps disconnecting randomly or drops the connection after a few minutes.
- The device is unable to connect to the Wi-Fi network despite entering the correct password.

## Steps to Troubleshoot:

1. **Restart the Router and Modem**:
   - Unplug the **router** and **modem** for 30 seconds, then plug them back in. This can resolve many connection issues by refreshing the network.
   - Wait a few minutes for the devices to reboot and check if the Wi-Fi is now working.

2. **Check Wi-Fi Connection on Other Devices**:
   - Test the Wi-Fi on other devices like a smartphone or another laptop to see if the problem is specific to one device or if it’s affecting the entire network.

3. **Run the Network Troubleshooter**:
   - Right-click the **Wi-Fi icon** in the taskbar and select **Troubleshoot problems**.
   - Windows will automatically detect and attempt to fix common network issues.

4. **Restart the Computer**:
   - Sometimes, restarting the computer can resolve temporary issues related to network connectivity.

5. **Check for Disabled Network Adapter**:
   - Press `Windows + X` and select **Device Manager**.
   - Expand the **Network Adapters** section and check if the Wi-Fi adapter is disabled (it will have a small arrow next to it).
   - Right-click on the adapter and select **Enable** if it is disabled.

6. **Update the Wi-Fi Adapter Driver**:
   - In **Device Manager**, right-click on the Wi-Fi adapter and select **Update Driver**.
   - You can choose **Search automatically for updated driver software**, or go to the manufacturer’s website to download the latest drivers.

7. **Forget and Reconnect to the Network**:
   - Go to **Settings > Network & Internet > Wi-Fi > Manage Known Networks**.
   - Select your Wi-Fi network, click **Forget**, and then reconnect by selecting it again from the available networks list. Re-enter your password when prompted.

8. **Check Wireless Settings on the Router**:
   - Log in to the router’s web interface (usually accessible by typing `192.168.1.1` or `192.168.0.1` in the web browser).
   - Check that the **Wi-Fi settings** are correct and that the router is broadcasting the network properly.
   - Ensure that the router is using an appropriate frequency band (2.4 GHz or 5 GHz).

9. **Reset Network Settings**:
   - Go to **Settings > Network & Internet > Status**, and scroll down to **Network reset**.
   - Click **Reset now**. This will remove and reinstall all network adapters and reset networking components to their default settings.
   - **Warning**: This will remove all saved Wi-Fi networks and passwords.

10. **Check for Interference**:
   - Move closer to the router to eliminate physical obstructions (walls, large objects).
   - Avoid interference from other wireless devices like microwaves, cordless phones, or baby monitors, which can affect Wi-Fi signal strength.

## Fix:
After following the above steps, the Wi-Fi connectivity issue was resolved by updating the Wi-Fi driver and restarting the router. Once the adapter was re-enabled and the network settings were reset, the system was able to reconnect to the Wi-Fi without any further issues.
