# Printer Not Working

## Symptoms:
- The printer is not responding or not printing any documents.
- The printer is offline or showing as "Paused" or "Error."
- Documents are stuck in the print queue and are not printing.
- Printer is not detected by the system or appears as "Not Connected."

## Most Important Fixes:

1. **Restart the Printer and Computer**:
   - **Restart** both the printer and your computer to reset any communication issues.
   - Turn off the printer, wait for 30 seconds, and then turn it back on. Restart your computer to clear any temporary issues.
   - After restarting, check if the printer is now working.

2. **Check Printer Connections**:
   - If you're using a **wired printer**, check that the printer's USB cable is securely connected to the computer. Try using a different USB port if possible.
   - If you're using a **wireless printer**, ensure the printer is connected to the same Wi-Fi network as your computer.
   - Verify the **Wi-Fi signal strength** near the printer to ensure a stable connection.

3. **Run the Printer Troubleshooter**:
   - Windows has a built-in **Printer Troubleshooter** that can help detect and resolve common printer issues.
   - Go to **Settings > Update & Security > Troubleshoot > Additional troubleshooters > Printer** and run the troubleshooter.
   - It will automatically check for issues like outdated drivers, paper jams, or connection problems.

4. **Check Printer Status and Set as Default Printer**:
   - Open **Control Panel > Devices and Printers**.
   - Right-click on the printer and select **See what's printing**.
   - If you see any documents in the print queue, cancel or delete them.
   - Ensure your printer is set as the **default printer** by right-clicking on the printer and selecting **Set as default printer**.

5. **Update Printer Drivers**:
   - Outdated or corrupted **printer drivers** can prevent the printer from working properly.
   - Go to **Device Manager** (right-click the **Start** button > **Device Manager**), expand **Printers**, right-click your printer, and select **Update driver**.
   - You can either search for the latest driver automatically or visit the printer manufacturer's website to download and install the latest driver.

6. **Clear the Print Queue**:
   - Sometimes, print jobs get stuck in the print queue and prevent new jobs from being processed.
   - Open **Control Panel > Devices and Printers**, right-click the printer, and select **See what's printing**.
   - Cancel all pending print jobs in the queue.
   - You may need to restart the print spooler service:
     - Open **Command Prompt as Administrator** and run:
       ```bash
       net stop spooler
       net start spooler
       ```

7. **Reinstall the Printer**:
   - If none of the above steps work, try **reinstalling** the printer.
   - Go to **Control Panel > Devices and Printers**, right-click your printer, and select **Remove device**.
   - After removing the printer, restart your computer and reinstall the printer using the installation CD or by downloading the latest drivers from the printer manufacturer’s website.

## Fix:
After restarting both the printer and the computer, and clearing the print queue, the printer started working again. **Updating the printer drivers** also helped resolve any outdated driver issues. Running the **Printer Troubleshooter** identified and fixed minor configuration problems, ensuring smooth printing thereafter.
