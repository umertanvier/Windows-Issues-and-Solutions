# Blue Screen of Death (BSOD)

## Symptoms:
- The system crashes and displays a blue screen with an error message or error code like **STOP 0x0000007B**, **0x0000001E**, etc.
- The system restarts automatically after showing the BSOD, or it remains stuck on the blue screen.
- You may also see messages like "Critical_Process_Died" or "Page_Fault_In_Non_Paged_Area."

## Steps to Troubleshoot:

1. **Note the Error Code on BSOD**:
   - On the BSOD screen, note down the error code and message displayed. This will help identify the root cause of the problem.
   - Common error codes include **0x0000007B** (inaccessible boot device), **0x0000001E** (KMODE_EXCEPTION_NOT_HANDLED), etc.

2. **Boot into Safe Mode**:
   - Restart the system and press **F8** or **Shift + F8** (depends on system configuration) to enter **Advanced Boot Options**.
   - Select **Safe Mode** or **Safe Mode with Networking** to boot into a minimal version of Windows.

3. **Check for Recent Changes**:
   - If BSOD appeared after a recent software or hardware change, **uninstall** or **disable** the new software/hardware and check if the issue resolves.
   - Go to **Control Panel > Programs** and uninstall any recently installed programs.

4. **Run System File Checker (SFC)**:
   - Open **Command Prompt** as Administrator (right-click on **Start** button and select **Command Prompt (Admin)**).
   - Run the following command:
     ```bash
     sfc /scannow
     ```
   - This will check and repair any corrupted system files that may be causing the issue.

5. **Run DISM Tool**:
   - Open **Command Prompt** as Administrator.
   - Run the following command to repair the system image:
     ```bash
     dism /online /cleanup-image /restorehealth
     ```
   - Wait for the process to complete, and then restart the system.

6. **Update Device Drivers**:
   - Go to **Device Manager** (right-click **Start** button and select **Device Manager**).
   - Right-click on devices (especially **display** and **storage drivers**) and select **Update Driver**.
   - You can also visit the manufacturer's website to download and install the latest drivers.

7. **Check for Hardware Issues**:
   - **RAM**: Run a **Memory Diagnostic Tool** (search for **Windows Memory Diagnostic** in Start Menu) to check for memory issues.
   - **Hard Drive**: Run **CHKDSK** to check for disk errors:
     ```bash
     chkdsk /f /r
     ```

8. **Rollback to a Previous Restore Point**:
   - If the issue started recently, perform a **System Restore**:
     - Go to **Control Panel > System and Security > System > System Protection**.
     - Select **System Restore** and choose a restore point before the issue started.

9. **Check for Overheating or Power Issues**:
   - Sometimes, BSOD can be caused by **overheating** or **power issues**. Check that the cooling system is working properly (fans and heat sinks).
   - Ensure that the power supply unit (PSU) is functioning correctly.

## Fix:
After following the troubleshooting steps, the system BSOD issue was resolved by updating the device drivers and running **DISM** and **SFC** scans. If the error was related to hardware, running a **memory check** or **disk check** helped identify and resolve the underlying problem.
